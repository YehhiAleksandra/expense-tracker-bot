## О проекте

Telegram-бот для учёта расходов по системе **«4 конверта»** с синхронизацией в Notion.

### Возможности

- 4 недельных конверта, 9 категорий
- Быстрый ввод: `е 15.5 обед` и `+500 премия`
- Inline-меню: /budget, /week, /undo
- Notion + SQLite

### Установка

```bash
git clone https://github.com/YehhiAleksandra/expense-tracker-bot.git
cd expense-tracker-bot
pip install python-dotenv requests
cp .env.example .env
python3 bot.py
```

## About

Telegram bot for personal expenses using the **4 envelopes** budgeting method with Notion sync.

### Features

- Weekly spending envelopes, 9 categories
- Quick input: `food 12.5 lunch`, `+500 bonus`
- Inline menu, `/budget`, `/week`, `/undo`
- Notion cloud sync + SQLite backup

### Quick start

```bash
git clone https://github.com/YehhiAleksandra/expense-tracker-bot.git
cd expense-tracker-bot
pip install python-dotenv requests
cp .env.example .env
python3 bot.py
```
